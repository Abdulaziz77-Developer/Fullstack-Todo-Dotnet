# Minimal API ToDo Project
This is a simple ToDo application built with ASP.NET Core Minimal API.
It supports basic CRUD operations for managing ToDo items.
## Features
- Create, Read, Update, Delete ToDo items
- In-memory data storage
- Swagger UI for API testing
- Request logging via middleware
## Requirements
- .NET 9 (or .NET 8)
- Any IDE (VS Code, Visual Studio)
## How to run
1. Clone the repository
2. Navigate to project folder
3. Run `dotnet restore`
4. Run `dotnet run`
5. Open Swagger UI at http://localhost:5000/swagger
## API Endpoints
- GET /todos
- GET /todos/{id}
- POST /todos
- PUT /todos/{id}
- DELETE /todos/{id}
